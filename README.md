# Template Node Typescript Lint Pretttier Jest Supertest

## Setup

```
mkdir <folder-project>

cd <folder-project>

git clone https://github.com/marcusvbrangel/template-node-typescript-jest.git .

np install

npm run build:tsc

npm run lint:check

npm run lint:fix

npm run style:check

pm run style:fix

node dist/src/server.js

npm run test

npm run test:watch

npm run test:coverage

```
